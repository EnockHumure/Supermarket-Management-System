## 🚀 Getting Started

### System Requirements
- **Operating System:** Windows 7 or higher
- **Database:** PostgreSQL 9.6 or higher
- **Java:** JDK 1.8 or higher
- **Memory:** 4GB RAM minimum
- **Storage:** 500MB for application + database space

### Installation Process
1. **Database Setup** (15 minutes)
   - Install PostgreSQL
   - Create database
   - Configure connection

2. **Server Installation** (10 minutes)
   - Deploy server application
   - Configure settings
   - Start services

3. **Client Installation** (5 minutes per workstation)
   - Install client application
   - Configure server connection
   - Create user accounts

4. **Training** (2-4 hours)
   - Admin training
   - Manager training
   - Cashier training

**Total Setup Time:** 1-2 days for complete deployment
